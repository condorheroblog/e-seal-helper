## 字体选择说明

根据[中国印章行业团体标准 T/CYZIA005-2017](http://cdyzxh.com/upload/file/2019-01-10/6368271206957793817692837.pdf) 圆形印章汉字使用简化字，字体为**宋体**，字母和数字字体为 **Arial**。椭圆形的印章（例如发票专用章）汉字使用简化字，字体为**仿宋体**，字母和数字字体依然是 **Arial**。
